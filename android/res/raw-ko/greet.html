왜 4초후에 오디오가 들리는가?

이것은 사용하는 브라우저의 종류와 버퍼링 때문에 발생합니다.

해결방법:

  • 플래쉬 렌더러를 사용. 이것은 지연방지를 위해 만들어진 것임.
  • 파이어폭스와 Opus 오디오를 사용. Opus 는 적은 지연과 밴드폭을 사용하는 오디오 스트리밍 코덱임. 파이어폭스는 적은 버퍼링을 사용함.
  • VLC 미디어 플레이어를 사용. 메뉴에서 Media ⇨ 네트워크 스트림 열기에서 http://your-ip:your-port/audio.wav 로 열고, "고급설정" 으로 들어가서 버퍼링을 1mS 로 설정함.

태스크 이벤트 제어

Tasker는 안드로이드 스마트폰을 위한 자동화 솔루션입니다. 문자를 보내거나 위치에 따라 착신벨소리를 바꾸거나 스케쥴에 따라 움직이게 해 줍니다.

IP Webcam Pro 버젼은 사용자 정의 UI 웹인터페이스나 태스커 이벤트를 만들 수 있습니다.

IP Webcam 이벤트를 사용해서 태스커를 설정하려면 "+" 버튼을 눌러 새 프로파일을 만들고, 상태 ⇨ 플러그인 ⇨ IP Webcam 트리거를 선택하세요.

웹인터페이스에서 만들어진 이벤트는 버튼텍스트와 같은 이벤트 이름을 가집니다. 예를 들면 가운데 버튼은 "5" 번 이벤트로 만들어 집니다.
