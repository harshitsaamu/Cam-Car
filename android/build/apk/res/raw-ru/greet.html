Звук запаздывает на 4 секунды?

Это происходит из-за внутренней буфферизации браузера.

Попробуйте эти решения:

  • Используйте встроенный проигрыватель Flash. Он сделан для уменьшения запоздания.
  • Используйте браузер Firefox и проигрыватель Opus. Opus — высококачественный кодек следующего поколения, разработанный для низкой латентности и маленького размера. Firefox почти не буфферизирует его.
  • Прослушивайте аудио через VLC Media Player. В его меню выберите Медиа ⇨ Открыть сетевой поток, выберите http://ваш-ip:ваш-порт/audio.wav как URL, выберите "Показывать больше настроек" и установите буферизацию в 1 мс.

События Tasker

Tasker — система автоматизации смартфонов на Android. Она может автоматизировать, скажем, отправку SMS или менять громкость звонка по расписанию или по местоположению.

IP Webcam Pro может генерировать события для Tasker, либо через веб-интерфейс или через редактируемый интерфейс.

Чтобы настроить Tasker для работы с событиями и состояниями IP Webcam, создайте в Tasker профиль, нажав кнопку "+" и выбрав State ⇨ Plugin ⇨ IP Webcam как условие.

События, сгенерированные из веб-интерфейса, имеют то же имя, что написано на кнопке. Скажем, центральная генерирует событие с именем "5".
