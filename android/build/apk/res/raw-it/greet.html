Perché sento l’audio 4 secondi in ritardo?

Questo accade a causa del browser buffering e dipende dal browser utilizzato.

Prova le seguenti soluzioni:

  • Usa renderer Flash. È progettato per un ritardo ridotto.
  • Utilizzare Firefox e Opus audio. Opus è il next-gen a bassa latenza bassa larghezza di banda con codec audio streaming e Firefox fa poco buffer di esso.
  • Usare VLC Media Player. Nel menù, aprire Media ⇨ Open Network Stream, immettere http://tuo-ip:tuo-port/audio.wav come URL, fare clic su "Mostra più opzioni" e impostare il buffering a 1 ms.

Tasker controllo eventi

Tasker è una soluzione di automazione per gli smartphone Android. Egli può, per esempio, inviare un SMS o variare il volume della suoneria in base alla tua esigenza o alla preventiva pianificazione.

IP Webcam Pro può generare eventi Tasker, sia con interfaccia web o con interfaccia utente personalizzabile.

Per configurare Tasker per l'utilizzo di eventi IP Webcam, creare un nuovo profilo utilizzando il pulsante "+" e selezionare Stato ⇨ Plugin ⇨ IP Webcam come trigger.

Eventi generati da interfaccia web hanno lo stesso nome di evento come il testo del pulsante. Ad esempio, il tasto centrale genera l'evento denominato "5".
